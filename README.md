# Slim Graph

A web-based tool for creating node-link diagrams and network visualizations from CSV data.

## ⚠️ Disclaimer

**This is a demo/experimental project.**
Use at your own risk. No warranties or guarantees of any kind are provided.
The author is not responsible for any issues arising from use of this software.

## Usage

1. Open `index.html` in a browser
2. Upload a CSV file with your graph data
3. View and interact with the generated visualization
